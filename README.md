 # AWS CDK Deployer

 A docker image contains AWS deployment tools.

 Tools:

 * NodeJS 22
 * Python 3.13
 * AWS CLI: `2.22.5`
 * AWS SAM CLI: `v1.131.0`
 * eksctl: `v0.194.0`
 * Helm: `v3.16.3`
 * Istioctl: `1.24.1`
 * K9s: `v0.32.7`
 * Kubectl: `v1.31.3`
 * Kustomize: `5.5.0`
 * Rain: `v1.19.0`
 * Support for Docker-in-Docker

 ## Setup

 Run `make up` from app root directory
